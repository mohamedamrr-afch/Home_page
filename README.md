# Homepage — CS50W Project 0

## Overview
Homepage is a simple multi-page personal website created as part of **CS50W Project 0**.  
It demonstrates basic web-development skills using **HTML**, **CSS**, and optional **JavaScript**.  
The site includes multiple pages, internal navigation, responsive layout, and custom styling.

---

## Features
- ✔️ Clean, multi-page personal website  
- ✔️ Navigation bar linking all pages  
- ✔️ Responsive design using CSS  
- ✔️ Organized page structure and semantic HTML  
- ✔️ Custom fonts, colors, and styles  
- ✔️ At least **four** required pages:
  - `index.html`
  - `about.html`
  - `hobbies.html`
  - `contact.html`
- ✔️ Use of:
  - CSS selectors and classes  
  - At least one Bootstrap component (optional if used)  
  - Images and external links  
  - Flexbox or Grid (optional but recommended)
